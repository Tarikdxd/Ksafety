KSafety

Kanser INC. tarafından yapıldı.

Windows uygulama güvenliği.

Başlangıç:
1. KSafety.exe dosyasını çalıştır.
2. İlk açılışta PIN belirle.
3. Korunan uygulamalar bölümünden .exe seç.
4. Zaman kuralları bölümünden kilit saatlerini ayarla.
5. Hızlı ayarlar bölümünden davranışları aç/kapat.

Öne çıkanlar:
- Daha dengeli ve kurumsal north-dark arayüz
- Stabil custom buton çizimi, daha az flicker/kaybolma riski
- Daha geniş ve orantılı pencere yerleşimi
- Koruma kapatma PIN doğrulamalı
- 15 dakika duraklatma PIN doğrulamalı
- Tray veya uygulama çıkışı PIN doğrulamalı
- Windows ile otomatik başlatma
- Her geri dönüşte PIN sor seçeneği
- Uzak kalınca açık uygulamayı kapatmadan kilitleme
- Dakika bazlı süre özelleştirme
- Ayar klasörüne hızlı erişim

Dağıtım:
Son kullanıcıya Release klasöründeki KSafety.exe verilebilir.
